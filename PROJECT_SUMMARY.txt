PROJECT OVERVIEW
================================================================================

AgentForge is a production-ready, 100% FREE multi-agent AI framework that enables:
- Autonomous task decomposition and delegation
- Multi-agent collaboration with specialized roles
- Advanced reasoning strategies (ReAct, Tree-of-Thought, Chain-of-Thought)
- Local model execution via Ollama or HuggingFace (no API keys needed)
- Optional OpenAI/compatible API integration

LLM PROVIDER OPTIONS
================================================================================

OPTION 1: MOCK (NO MODEL - WORKS IMMEDIATELY)
- Uses MockLLMProvider
- Realistic mock responses for all problem types
- Perfect for testing and development
- Command: python examples/01_simple_task.py

OPTION 2: OLLAMA (LOCAL, FREE)
- ollama pull mistral (or llama2, codellama, etc.)
- ollama serve
- export AGENTFORGE_OLLAMA_MODEL=mistral
- No API key needed, fully local

OPTION 3: HUGGINGFACE TRANSFORMERS (LOCAL, FREE)
- pip install transformers torch
- export AGENTFORGE_HF_MODEL=mistralai/Mistral-7B-Instruct-v0.1
- Runs on CPU or GPU, fully local

OPTION 4: OPENAI API (OPTIONAL, PAID)
- export OPENAI_API_KEY=sk-...
- Optional integration, not required
- Also works with compatible APIs (LM Studio, vLLM)

================================================================================
KEY DESIGN DECISIONS
================================================================================

1. ASYNC/AWAIT
   - Enables concurrent agent execution
   - Allows parallel task processing
   - Better resource utilization

2. TOOL-BASED ARCHITECTURE
   - Agents can request tools, not just generate text
   - ReAct-style reasoning with action selection
   - Flexible and extensible

3. SHARED MEMORY
   - Agents can learn from each other
   - Context retention across turns
   - Relevance-based retrieval

4. SPECIALIZED AGENTS
   - Each agent has a specific role
   - Different reasoning strategies
   - Easy to extend with new agents

5. FREE LOCAL MODELS FIRST
   - MockLLMProvider for testing
   - Ollama for easy local setup
   - Transformers for maximum flexibility
   - OpenAI as optional upgrade

6. SECURITY FIRST
   - Sandboxed code execution
   - Dangerous operations blocked
   - Whitelisted shell commands
   - Safe expression evaluation
